# JSONPlaceholder API tests

Проект содержит простые API-тесты для проверки эндпоинта `/posts`
сервиса JSONPlaceholder.

## Что проверяется

- получение списка постов;
- получение поста по id;
- обработка несуществующего поста;
- создание поста;
- обновление поста;
- удаление поста.

## Важный момент

JSONPlaceholder — это тестовый API. Он имитирует создание, обновление
и удаление данных, но не сохраняет изменения на сервере.

Поэтому в тестах POST, PUT и DELETE проверяется ответ API,
а не дальнейшее наличие изменённых данных.
